# Installation

Injectez de la puissance dans vos scripts Fluxer avec une installation simple.

## Prérequis

- [Bun](https://bun.sh) (recommandé) ou Node.js v18+
- Un compte Fluxer.app

## Installation via Bun

```bash
bun add fluxer-selfbot.js
```

## Utilisation directe

Si vous avez cloné le dépôt :

```bash
cd librairies
bun install
bun run build
```
